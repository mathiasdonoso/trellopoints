# Trellopoints

A simple webhook for story cards in trello

## Getting Started

```
go get github.com/mathiasdonoso/trellopoints
```

## Configuration

### Environment variables needed

* TRELLO_KEY
* TRELLO_TOKEN

## Tests

To run the tests

```
go test
```
